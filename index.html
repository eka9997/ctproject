비버의 자동 분류 시스템 - 숫자 규칙 추론하기

대상 및 교과: 중3 수학, 정보

지문: 아래 제시된 분류기는 숫자를 입력하면 A, B, C 그룹 중 하나로 분류해 줍니다. 이 문제는 컴퓨팅 사고의 과정을 통해 해결할 수 있습니다. 컴퓨팅 사고는 다음과 같은 사고 과정을 기반으로 문제를 해결하는 방식입니다:
  • 문제 분해: 현재 상태와 목표 상태로 문제를 나누어 보기
  • 추상화: 핵심적인 수학적 요소만 남기기
  • 패턴 인식: 숫자 간의 공통 규칙 또는 관계 찾기
  • 알고리즘: 규칙에 따라 조건을 만들고 명확한 절차로 분류하기
규칙:
  • A 그룹: 6의 배수
  • B 그룹: 등차수열 (초항 2, 공차 6) → 2, 8, 14, 20, 26, ...
  • C 그룹: 3 × 홀수 → 3, 9, 15, 21, 27, ...
숫자를 입력하고 시간 제한을 설정한 후 '문제 풀기' 버튼을 눌러 분류해 보세요.

⏰ 시간이 종료되었습니다!
